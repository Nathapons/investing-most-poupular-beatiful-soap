
# Investing.com: Most Popular News by Beautiful Soup

Get Most Popular News by Python

## Prerequisites

*   **Python 3.12+**
## Data Resource

[Most Popular Financial News on Investing.com](https://www.investing.com/news/most-popular-news)
## Acknowledgements

 - [cloudscraper](https://docs.python-requests.org/en/latest/user/quickstart/)
 - [BeautifulSoup](https://www.crummy.com/software/BeautifulSoup/bs4/doc/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Nathapons/investing-most-poupular-beatiful-soap.git
```

Go to the project directory

```bash
  cd investing-most-poupular-beatiful-soap
```

Activate Python Environment

```bash
  python -m venv env
```

Install Python Library

```bash
  source env/bin/activate
  pip install -r requirements.txt
```

Start Source Code

```bash
  python main.py
```
